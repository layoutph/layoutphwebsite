Welcome to the BitcoinTalk Public Information Project (BPIP)

This project aims to scrape all publically available information to see what type of fun stats we can come up with.
We start with the 937,778 registered accounts and delete the 649,712 accounts that have not signed on for 90 days, have no posts and no trust. This leaves us with 288,066 active accounts.

We currently scrape the public https://bitcointalk.org/index.php?action=profile page, with an account that uses only default trust settings. All the information you see has been gleemed from that one page.

In the future we plan to add trending (for example, so you can see how an account's trust score moves over time) and other neat features.
If you have a statistic or idea you'd like to see, please email vod@martinlawrence.ca
